# Principe des corrections

Les corrections sont difficiles et délicates.

## Pricipes généraux

### Paiements des Commandes et des Factures

Il est uniquement possible de supprimer un paiement d'une commande.

Il n'est pas possible de modifier le montant. Il faut supprimer le paiement et en créer un nouveau.

Pour ce faire, aller dans l'administration du magasin, choisir l'option dangereuse: Paiements - Manipulation directes,
rechercher la commande via son numéro, et supprimer le paiement.

Pour ajouter un paiement à une commande, utiliser la procédure normale: Visualiser la commande,
choisir Paiements et entrer les données du paiement.

Il n'est pas possible d'ajouter un paiement à une commande complètement payée.
